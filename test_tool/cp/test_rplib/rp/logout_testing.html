Logout RP Profile

A set of Logout RP test is currently available in pilot phase. The tests can be grouped into:

rp-init-logout
Tests RP initiated logout as described in RP-Initiated Logout.
rp-backchannel-rpinitlogout
Tests RP initiated logout together with Back-Channel logout
rp-frontchannel-rpinitlogout
Tests RP initiated logout together with Front-Channel logout

As can be understood from the list above supporting RP initiated logout is a prerequisite to do any of these tests.

Furthermore, support for provider information discovery and dynamic client registration is also needed.

Running a test will consists of sending the following requests to the test suite (==OP)

  1. provider info discovery
  2. client registration
  3. authorization
  4. access token (depending on which response type is chosen)
  5. end session

Please test this test and submit your results to certification@oidf.org as described in Submission of Results for RPs
